## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Running and create the database

```bash
docker-compose up -d
```

## Prisma

```bash
# Setup
npm install -D prisma
npx prisma init
# // .env
# DATABASE_URL="postgres://myuser:mypassword@localhost:5432/median-db"


# Migrate DB
npx prisma migrate dev --name "init"

# Seed the data
touch prisma/seed.ts
# adding the prisma.seed key to the end of your package.json file
  # "prisma": {
  #   "seed": "ts-node prisma/seed.ts"
  # }
# then
npx prisma db seed


# Creaet Prisma Service
npx nest generate module prisma
npx nest generate service prisma

# Generate REST resources
npx nest generate resource
# This will create new Resource in the service

# You will be given a few CLI prompts. Answer the questions accordingly:

# What name would you like to use for this resource (plural, e.g., "users")? articles
# What transport layer do you use? REST API
# Would you like to generate CRUD entry points? Yes
```

[Link](https://www.prisma.io/blog/nestjs-prisma-rest-api-7D056s1BmOL0)

## Add apps

```bash
nest generate app upload
```

## Running the select app

```bash
yarn run {app_name} `(inside the apps)`
```

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
